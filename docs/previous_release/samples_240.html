Samples

The following samples showcase the ESL profile and service API usage on the Nordic’s boards:

  • Bluetooth: Central ESL

  • peripheral_esl_240

Note

To test ESL functionality, both samples must be used in pparallel: on one DevKit (e.g. nRF52833 DK) run the Bluetooth: Peripheral ESL, while on another DevKit (e.g. nRF5340 DK) run the Bluetooth: Central ESL.

In the ESL repository, all samples are placed in the samples directory.
